# Contributing to This Portfolio\n\nWhile this is a personal portfolio, I appreciate feedback and suggestions! Here's how you can contribute:\n\n## 🐛 Report Issues\n\nFound a bug or accessibility issue? Please report it:\n1. Check if the issue already exists\n2. Create a new GitHub Issue with:\n   - Clear description\n   - Steps to reproduce\n   - Screenshots if applicable\n   - Browser/device info\n\n## 💡 Suggestions\n\nHave ideas for improvements?\n- Open a GitHub Discussion\n- Suggest design improvements\n- Point out performance optimizations\n- Recommend content updates\n\n## 🔧 Technical Guidelines\n\nIf you're forking to create your own portfolio:\n\n### Code Style\n- Use semantic HTML\n- Maintain Tailwind utility-first approach\n- Keep JavaScript vanilla (no frameworks)\n- Ensure accessibility standards (WCAG 2.1)\n- Test on mobile devices\n\n### Performance\n- Minimize external requests\n- Use CDN-hosted libraries\n- Optimize animations with CSS transforms\n- Test Lighthouse scores\n\n### Accessibility\n- Test with keyboard navigation\n- Verify screen reader compatibility\n- Maintain proper color contrast ratios\n- Include aria-labels where needed\n- Support prefers-reduced-motion\n\n## 🚀 Deployment\n\nThis portfolio is deployed via GitHub Pages with automatic builds.\n\n## 📧 Contact\n\nFor detailed feedback, reach out:\n- Email: dogarasad277@gmail.com\n- LinkedIn: [asad-ullah-dogar](https://www.linkedin.com/in/asad-ullah-dogar/)\n- GitHub: [@Asadullah-Dogar](https://github.com/Asadullah-Dogar)\n\nThanks for your interest! 🎉\n"
